使用很少的代码快速构建一个漂亮的管理后台

简洁、易用、高效


<?php

class UserController extends Controller
{
    /**
     * Make a grid builder.
     * @return  Grid
     */
	public function grid()
	{
        $grid = new Grid(new Administrator);

        $grid->id()->sortable();
        $grid->username();
        $grid->name();
        $grid->roles()->pluck('name')->label('primary');
        $grid->created_at();

        return $grid;
	}
}

特性
简洁易用的语义化API

Dcat Admin提供简洁易用的语义化接口让开发者可以通过很少的代码就可以构建出非常完善的后台系统。

独立的数据操作接口

Dcat Admin把对数据的CURD操作抽象成简单清晰的接口，构建页面不再需要关心数据操作接口的具体实现，从而可以轻松实现对不同数据源的支持。
一键生成CURD页面

支持通过页面自定义、以及根据已有数据表一键生成CURD代码。可生成文件包括：控制器、模型、翻译文件、表迁移文件、IDE提示文件等。
支持pjax按需加载

Dcat Admin通过pjax构建单页应用，并且支持按需加载前端资源，开发者无需担心安装的组件过多而影响页面加载速度。
内置丰富实用的页面组件

内置近100种实用的页面组件，其中包括40+表单组件，涵盖了大部分业务场景，并且支持扩展自定义组件。

内置权限系统

内置可扩展RBAC权限系统，并且方便和路由、菜单绑定。
支持Ajax提交表单

支持Ajax提交表单，仅用数行代码即可构建出Ajax表单弹窗。

内置实用上传组件

支持分片上传、图片预览、上传进度条，支持所有OSS平台。
支持双表头数据表

支持双表头数据表，可轻松构建数据报表。
